# Shop
간단 쇼핑몰 예제

사용
<ul>
<li>SpringBoot</li>
<li>Thymeleaf</li>
<li>H2database</li>
<li>SpringDataJpa</li>
<li>SpringSecurity</li>
<li>QueryDSL</li>
</ul>

구현 기능
<ul>
<li>Entity C,R,U,D</li>
<li>회원가입 등에 대한 Validate 이용 유효성 검증</li>
SpringSecurity
<ul>
<li>권한에 따른 행동제약</li>
<li>관리자만 문의에 답변할 수 있게끔 처리</li>
<li>Secure 사용</li>
</ul>
<li>비 동기식 리뷰 처리</li>
<li>구매한 상품에 대해서만 리뷰를 작성할 수 있게끔 처리</li>
<li>페이징</li>
<li>검색</li>
<li>Entity를 직접 노출하거나 받지않고 Request, Response로 나누어 사용</li>
</ul>
